Expressions and Conditions

In programming, we use expressions to calculate values and conditions to make decisions. We’ll focus on the following comparison operators: <, >, =, ==, and ===.

Examples of Expressions

Expressions involve comparing or calculating values. Some common examples include:

// Using comparison operators:
let age = 20;
let isAdult = age > 18; // true: age is greater than 18

let temperature = 25;
let isHot = temperature >= 30; // false: temperature is less than 30

let isSame = 5 == '5'; // true: values are equal (loose comparison)
let isStrictSame = 5 === '5'; // false: values and types must match (strict comparison)

Examples of Conditions

Conditions, like if and else, allow programs to make decisions based on expressions. Here’s an example:

// Deciding what to wear based on temperature
let temperature = 22;
if (temperature > 25) {
    console.log("Wear light clothes");
} else {
    console.log("Wear a jacket");
}

// Checking if a number is even or odd
let number = 7;
if (number % 2 === 0) {
    console.log("The number is even");
} else {
    console.log("The number is odd");
}
