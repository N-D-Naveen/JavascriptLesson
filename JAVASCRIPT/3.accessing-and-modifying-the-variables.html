Accessing and Updating Data Types

Understanding how to work with different data types in JavaScript is crucial for any programmer. Let’s explore examples of accessing and updating numbers, strings, booleans, objects, and arrays!

Examples

1. Numbers

You can simply reassign a number:

let age = 25; // Initial number
age = 30; // Updated number
console.log(age); // Output: 30

2. Strings

Strings can be modified by assigning a new value:

let name = "John"; // Initial string
name = "Doe"; // Updated string
console.log(name); // Output: "Doe"

3. Booleans

Booleans can be toggled easily:

let isOnline = true; // Initial boolean
isOnline = false; // Updated boolean
console.log(isOnline); // Output: false

4. Objects

Access and update specific properties of objects:

let user = { name: "Alice", age: 28 }; // Initial object
user.age = 29; // Update the age property
console.log(user); // Output: { name: "Alice", age: 29 }

5. Arrays

Access and modify array elements using their index:

let fruits = ["apple", "banana", "cherry"]; // Initial array
fruits[1] = "orange"; // Update the second element
console.log(fruits); // Output: ["apple", "orange", "cherry"]
