JavaScript Loops and Array Methods

1. For Loop

A for loop is used when you know how many times you want to run a block of code.

    // Example of a for loop
    let numbers = [1, 2, 3, 4, 5];
    for (let i = 0; i < numbers.length; i++) {
        console.log(numbers[i]);
    }

Output: The loop will log each number in the array.

2. For...In Loop

The for...in loop is used to iterate over the properties of an object.

    // Example of a for...in loop
    let person = { name: "John", age: 30, city: "New York" };
    for (let key in person) {
        console.log(key + ": " + person[key]);
    }

Output: The loop will log each property and its value.

3. Array Methods: Map, Filter, and Reduce

3.1. Map

The map() method creates a new array populated with the results of calling a provided function on every element in the array.

    // Example of map
    let numbers = [1, 2, 3, 4];
    let squaredNumbers = numbers.map(num => num * num);
    console.log(squaredNumbers); // Output: [1, 4, 9, 16]

3.2. Filter

The filter() method creates a new array with all elements that pass the test implemented by the provided function.

    // Example of filter
    let numbers = [1, 2, 3, 4, 5];
    let evenNumbers = numbers.filter(num => num % 2 === 0);
    console.log(evenNumbers); // Output: [2, 4]

3.3. Reduce

The reduce() method executes a reducer function (that you provide) on each element of the array (from left to right) to reduce it to a single value.

    // Example of reduce
    let numbers = [1, 2, 3, 4];
    let sum = numbers.reduce((total, num) => total + num, 0);
    console.log(sum); // Output: 10
