Basic Operators in JavaScript

Examples of Basic Operators

In JavaScript, there are several basic operators that are commonly used:

1. Arithmetic Operators

These operators perform mathematical operations on numbers:

    // Addition
    let sum = 5 + 3;
    console.log(sum);  // Output: 8

    // Subtraction
    let difference = 5 - 3;
    console.log(difference);  // Output: 2

    // Multiplication
    let product = 5 * 3;
    console.log(product);  // Output: 15

    // Division
    let quotient = 6 / 3;
    console.log(quotient);  // Output: 2

2. Comparison Operators

These operators compare values and return a boolean result (true or false):

    // Equal to
    let isEqual = 5 == 3;
    console.log(isEqual);  // Output: false

    // Greater than
    let isGreaterThan = 5 > 3;
    console.log(isGreaterThan);  // Output: true

    // Less than
    let isLessThan = 5 < 3;
    console.log(isLessThan);  // Output: false

3. Logical Operators

These operators combine multiple conditions:

    // AND operator (both conditions must be true)
    let isAdult = true;
    let hasID = true;
    let canBuyAlcohol = isAdult && hasID;
    console.log(canBuyAlcohol);  // Output: true

    // OR operator (either condition can be true)
    let hasTicket = false;
    let isVIP = true;
    let canEnter = hasTicket || isVIP;
    console.log(canEnter);  // Output: true
