Examples

Access Elements by ID

const appleBasket = document.getElementById("appleBasket");
console.log(appleBasket.innerHTML);

Access Elements by Class Name

const fruitBaskets = document.getElementsByClassName("fruitBasket");
console.log(fruitBaskets.length);

Change Text Content

const firstItem = document.querySelector("#bananaBasket li");
firstItem.textContent = "Updated Banana";

Add Class to Element

const appleBasket = document.getElementById("appleBasket");
appleBasket.classList.add("highlight");

Remove Class from Element

const appleBasket = document.getElementById("appleBasket");
appleBasket.classList.remove("highlight");

Update InnerHTML

const bananaBasket = document.querySelector("#bananaBasket");

    const inputBox = document.getElementById("fruitInput");

    const bananaItems = document.querySelectorAll("#bananaBasket li");
bananaItems.forEach(item => console.log(item.textContent));

    Documentation

    • Accessing Elements:
      • By ID: document.getElementById()
      • By Class: document.getElementsByClassName()
      • By Tag: document.getElementsByTagName()
      • By Selector: document.querySelector()
    • Properties:
      • innerHTML: Get or set HTML content.
      • textContent: Get or set plain text.
      • classList: Add, remove, or toggle classes.
      • style: Modify inline styles.
      • value: Get or set input values.
